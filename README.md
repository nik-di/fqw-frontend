### Установка
#### Git command:
1. Скачать репозиторий:
```git clone https://github.com/nik-di/fqw-frontend.git```
2. Перейти в директорию с каталогом
```cd fqw/```
3. Установить npm-зависимости:
```npm i```
4. Произвести сборку проекта:
```npm run build```
5. Запустить проект на локальном сервере:
```npm run dev```
